Bluetooth Audio Receiver

Objective
> Convert any wired speaker into a wireless Bluetooth audio device using a compact and rechargeable system.

📌 Overview
This project transforms a traditional wired speaker into a portable Bluetooth audio receiver using the XY-BT module.
It is powered by a 3.7V Li-Po battery with an integrated charging circuit for convenient daily use.

⚙️ Components
* XY-BT Bluetooth Audio Module
* Li-Po Battery (3.7V)
* TP4056 Charging Module
* Speaker (Left & Right)
* Power Switch

🎥 Demo
👉  https://www.youtube.com/watch?v=LJWp9kaEDrI

🔋 Power System
* Rechargeable Li-Po battery
* Charging via TP4056 module
* Power control using ON/OFF switch

🚀 Features
* 🔹 Wireless audio streaming via Bluetooth
* 🔹 Portable and rechargeable design
* 🔹 Simple and low-cost components
* 🔹 Easy to build and modify

⚠️ Limitations
* Audio output is relatively low without an amplifier
* Not suitable for high-power speakers

🛠️ Future Improvements
* Add audio amplifier (e.g., PAM8403)
* Improve enclosure design
* Add battery level indicator
